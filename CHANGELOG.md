# YeyeTour 업데이트 내역 (Changelog)

## [V2 Update] - 무료 API 연동 및 모듈화 개편

기존의 단일 파일 기반 및 하드코딩된 정적 데이터 구조에서 벗어나, 유지비용이 발생하지 않는 **무료 오픈소스 API 생태계 연동** 및 **구조적 모듈화**를 완료했습니다.

### 🚀 주요 변경 사항 (Key Changes)

1. **외부 API 모듈 분리 (`providers/` 패키지 도입)**
   - `server.py` 하나에 집중되어 있던 로직 중 외부 데이터 연동부를 `providers` 패키지로 분리하여 확장성과 가독성을 높였습니다.
   - **Nominatim (`nominatim.py`)**: OpenStreetMap 기반으로 관광지 및 도시의 정확한 위경도 좌표를 추출합니다.
   - **OSRM (`osrm.py`)**: 하버사인 직선거리를 넘어 실제 도로/도보망 기반의 이동 소요 시간(분)을 계산합니다.
   - **Wikipedia (`wikipedia.py`)**: 관광지의 역사/문화적 요약 설명을 실시간으로 동적 조회합니다.

2. **하드코딩(Mock) 데이터 제거 및 동적 탐색 적용**
   - 초기 데모를 위해 내장되어 있던 일부 이탈리아 도시(로마, 피렌체 등)의 정적 카탈로그를 제거했습니다.
   - 이제 전 세계 어느 도시나 관광지를 입력해도 실시간으로 API를 호출하여 대응할 수 있는 범용성을 확보했습니다.

3. **경로 추정의 정확도 향상**
   - 휴리스틱(단순 계산)에 의존하던 로직에서 **OSRM 엔진**이 결합되어 도보, 대중교통 등 이동 수단에 따른 더 현실적이고 세밀한 시간 계산이 가능해졌습니다.

4. **문서 현행화**
   - 위 구조적 변화 사항들을 반영하여 프로젝트의 기술 문서인 `DEVELOPMENT_SUMMARY.md`를 최신화했습니다.
